# Catalogo de fuentes oficiales

`sources.json` contiene las fuentes externas autorizadas para el MVP. El catalogo
es versionado, pero los PDF y HTML se descargan desde el organismo de origen.

Antes de usar una fuente fuera del ambito academico, revise sus condiciones de
uso. La aplicacion almacena URL, hash y fecha de descarga para facilitar la
trazabilidad; no convierte una guia en consejo clinico individual.
