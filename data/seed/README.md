# Datos ficticios

`wakyma_mock.json` simula una exportacion minima de Wakyma con 15 propietarios y
25 mascotas. No contiene informacion real. La carga principal del MVP se realiza
mediante `python -m app.seed`; el JSON queda como contrato de ejemplo para el
adaptador de integracion que se completara en una iteracion posterior.
