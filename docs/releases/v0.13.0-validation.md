# Validacion de GamuCare AI v0.13.0

## Resultado

- 64 pruebas de backend superadas.
- Cobertura global del backend: 59 %.
- Compilacion de Python correcta.
- Compilacion React, TypeScript y Vite correcta.
- Dataset de aceptacion actualizado para exigir los cuatro perfiles.
- Pruebas API que verifican el acceso tecnico y el bloqueo cruzado con las funciones de negocio.

## Casos especificos verificados

- `technical` puede consultar `/api/v1/observability/status`.
- `clinic` recibe HTTP 403 al intentar consultar el endpoint tecnico anterior.
- `technical` recibe HTTP 403 al intentar consultar `/api/v1/dashboard`.
- El esquema de usuario acepta el nuevo rol.
- La evaluacion formal requiere `clinic`, `staff`, `owner` y `technical`.
- La interfaz redirige al perfil tecnico a `/system-status`.
- React oculta las opciones tecnicas a los perfiles de negocio y viceversa.

## Limitaciones de validacion

No se ha levantado Docker Compose completo en este entorno. Deben comprobarse en Docker Desktop el inicio de sesion de la cuenta tecnica, la creacion idempotente del usuario y el acceso a los servicios de monitorizacion ya desplegados.
