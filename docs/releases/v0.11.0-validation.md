## Limitaciones de esta validacion

- No se ha levantado Docker Compose completo en el entorno de generacion.
- No se ha ejecutado Ollama con GPU ni una evaluacion real de VetIA.
- La compilacion completa de Vite se ejecutara al construir la imagen `web` en Docker Desktop o mediante GitHub Actions. En este entorno no fue posible completar `npm ci` por falta de acceso al registro de paquetes.
- La compatibilidad del script PowerShell se ha revisado para Windows PowerShell 5.1, pero su ejecucion final debe comprobarse en el equipo Windows de destino.
