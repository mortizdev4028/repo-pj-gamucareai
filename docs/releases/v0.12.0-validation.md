## No validado en este entorno

No se ha podido levantar Docker Compose porque el entorno de generacion no
dispone del daemon Docker. Por tanto, deben verificarse en Docker Desktop:

- Arranque real de exporters, Prometheus, Alertmanager y Grafana.
- Ejecucion completa de `backup.ps1` y `restore.ps1`.
- Compatibilidad final de los scripts con Windows PowerShell 5.1.
- Recuperacion de un snapshot Qdrant sobre los datos reales del usuario.

El script `upgrade-v0.12.ps1` incluye health checks y muestra logs si la API no
arranca. Tras actualizar se recomienda ejecutar:

```powershell
.\scripts\smoke-test.ps1 -IncludeMonitoring
.\scripts\backup.ps1
```
