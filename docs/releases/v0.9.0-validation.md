## Frontend

```text
npm ci
npm run build
TypeScript y Vite completados correctamente
```

Vite informa de un bundle principal superior a 500 kB. Es una advertencia de optimizacion, no un fallo de compilacion. Se tratara en una evolucion posterior mediante carga diferida de rutas.

`npm audit` informa de vulnerabilidades en dependencias transitivas del frontend. No se ha aplicado `npm audit fix --force` porque puede introducir cambios incompatibles sin revision. Se revisara dentro del bloque especifico de seguridad.

## Limitacion del entorno de validacion

No se ha ejecutado Docker Compose completo ni la migracion sobre PostgreSQL real en este entorno. El script de actualizacion ejecuta `alembic upgrade head` y las pruebas dentro del contenedor en el equipo Windows de destino.
